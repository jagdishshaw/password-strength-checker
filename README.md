# Password Strength Checker

A Python-based tool designed to evaluate the security level of passwords against cybersecurity compliance standards.

## 🔍 Features
* **Length Analysis:** Verifies if a password meets safe length requirements.
* **Character Variety Check:** Detects uppercase letters, lowercase letters, numbers, and special symbols.
* **Feedback System:** Explains exactly what is missing to make a weak password stronger.
